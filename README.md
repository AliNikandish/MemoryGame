## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/MemoryGame.git
```

#### Go to the project directory

```bash
  cd MemoryGame
```

#### Install MemoryGame with npm

```bash
  npm install
```
#### Run:
```bash
  npm run dev
```
